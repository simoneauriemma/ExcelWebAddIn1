## Quick start

Clone the repo: `git clone http://service.routetopa.eu:7480/WebCompDev/COMPONENTS.git`.

### What's included

Within the download you'll find the following directories and files. You'll see something like this:

```
COMPONENTS/
├── bower_components/
│   ├── ...
├── controllets/
│   ├── ...
│   └── ...
├── datalets/
│   ├── ...
│   └── ...
└── docs/
    ├── docs.html
```
